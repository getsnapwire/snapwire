Quick Start Guide

Step 1: Generate an API Key

Go to the API Keys tab and create a key for your agent.

Step 2: Send Tool Calls to Snapwire

Point your AI agent to use this endpoint before executing any tool:

POST /api/intercept
Authorization: Bearer af_YOUR_API_KEY
Content-Type: application/json

{
  "tool_name": "delete_file",
  "parameters": {"path": "/important.txt"},
  "intent": "Clean up old files",
  "agent_id": "my-agent",
  "webhook_url": "https://your-app.com/callback"
}

Step 3: Handle the Response

If the action is allowed, proceed. If blocked, the response includes an action_id you can poll for approval:

GET /api/actions/{action_id}
Authorization: Bearer af_YOUR_API_KEY

# Response: {"status": "approved"} or {"status": "pending"}

Step 4: Configure Your Rules

Go to the Rules tab to customize your safety rules, or install a template pack to get started quickly.

Python Example

import requests

API_KEY = "af_your_key_here"
FIREWALL_URL = "https://your-server.com"

def check_with_firewall(tool_name, params, intent=""):
    resp = requests.post(
        f"{FIREWALL_URL}/api/intercept",
        headers={"Authorization": f"Bearer {API_KEY}"},
        json={
            "tool_name": tool_name,
            "parameters": params,
            "intent": intent,
        }
    )
    result = resp.json()
    if result["status"] == "allowed":
        return True  # Safe to proceed
    elif result["status"] == "blocked":
        print(f"Blocked! Action ID: {result['action_id']}")
        return False  # Wait for human approval
    return result["status"] == "auto-approved"

# Usage
if check_with_firewall("send_email", {"to": "user@example.com"}):
    send_email(to="user@example.com")

🔗 Framework Integrations

from langchain.tools import Tool
from agentic_firewall import AgenticFirewall

fw = AgenticFirewall(api_key="YOUR_API_KEY", url="https://your-server.com")

class FirewallWrappedTool(Tool):
    def _run(self, query: str) -> str:
        result = fw.intercept(
            tool_name=self.name,
            parameters={"query": query},
            agent_id="langchain-agent"
        )
        if result["decision"] == "blocked":
            return f"Action blocked: {result['reason']}"
        return self.original_func(query)
