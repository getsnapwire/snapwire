Snapwire sends anonymous usage data (rule count, intercept count, uptime) to help improve the project. No rule content, agent names, or personal data is sent. You can disable this anytime in Settings.

Opt out: Set the DO_NOT_TRACK=1 environment variable, or toggle it off in Settings after setup.
