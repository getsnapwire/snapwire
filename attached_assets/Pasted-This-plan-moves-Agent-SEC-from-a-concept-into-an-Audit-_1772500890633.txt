#### **Prompt 2: The "Guardian" Identity Protocol (SEC-ID)**

> "Create a core module `identity.ts`. This module must generate a **Non-Fungible Security Passport (NFSP)** for any AI agent. Implement a middleware that rejects any incoming API request that does not have a valid, unexpired SEC-ID passport."

#### **Prompt 3: The NIST Compliance Engine**

> "Build a `compliance_engine.ts`. This module should ingest a JSON-based policy file (e.g., `nist_800_218_rules.json`). Implement an 'Active Auditor' function that monitors agent actions. If an agent attempts to call a function not listed in its `allowed_intent_scope`, the system must trigger a `CRITICAL_POLICY_VIOLATION` event, kill the process, and log the event to an immutable `audit_trail.log`."
